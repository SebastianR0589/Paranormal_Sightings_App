### Paranormal Sighting App
**Description:**  
This app is a simple input focused server app. The user can add new paranormal sightings which will be added to the displayed list.


**GitHub Repo:** [Link to repo](https://github.com/SebastianR0589/Paranormal_Sightings_App)
